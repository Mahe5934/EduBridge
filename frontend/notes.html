Python Syntax Basics

Python uses indentation to define code blocks instead of curly braces or keywords. This makes Python code highly readable.

# Example of Python indentation
if 5 > 2:
    print("Five is greater than two!")
    if 5 > 3:
        print("Five is also greater than three!")

Variables and Data Types

Python is dynamically typed, meaning you don't need to declare variable types explicitly.

# Variable assignment
x = 5           # Integer
y = "Hello"     # String
z = 3.14        # Float
is_valid = True # Boolean

Common Data Types:

  • int - Integer numbers (e.g., 5, -3, 42)
  • float - Decimal numbers (e.g., 3.14, -2.5)
  • str - Text strings (e.g., "Hello", 'World')
  • bool - Boolean values (True or False)

Basic Operations

Arithmetic Operators:

  • + - Addition
  • - - Subtraction
  • * - Multiplication
  • / - Division
  • // - Floor division
  • % - Modulus (remainder)
  • ** - Exponentiation
# Examples
result = 10 + 5     # 15
result = 10 / 3     # 3.333...
result = 10 // 3    # 3
result = 10 % 3     # 1

Control Structures

If Statements:

age = 18
if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

Loops:

# For loop
for i in range(5):
    print(i)  # Prints 0, 1, 2, 3, 4

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

Functions

Functions are defined using the def keyword:

def greet(name):
    return f"Hello, {name}!"

message = greet("Alice")
print(message)  # Output: Hello, Alice!

Lists

Lists are ordered collections that can contain different data types:

# Creating a list
fruits = ["apple", "banana", "cherry"]

# Accessing elements
first_fruit = fruits[0]  # "apple"

# Adding elements
fruits.append("orange")

# Length of list
count = len(fruits)

Best Practices

  • Use meaningful variable names
  • Comment your code for complex logic
  • Follow PEP 8 style guide for Python code
  • Test your code with different inputs
  • Handle errors gracefully with try/except blocks
